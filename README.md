# NLP Regex + Hanspell Check

### 1. Introduce 

- NLP 정규식
  - 자음, 모음, 특수기호 제거
  - HTML 태그 제거
  - " ! " n개 이상 1개로 치환
  - " ? " n개 이상 1개로 치환
- Hanspell Check
  - 네이버 한글 맞춤법 검사를 사용하기 위한 라이브러리
  - https://github.com/ssut/py-hanspell

### 2. Used

- > $ pip install -r requirements.txt
  >
- > $ git clone https://github.com/ssut/py-hanspell.git
  >

  - > $ cd py-hanspell
    >
  - > $ python setup.py install
    >
